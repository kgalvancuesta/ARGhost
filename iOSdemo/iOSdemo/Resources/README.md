# Resources Directory

## Professional Videos

Place your professional workout demonstration videos in the `ProfessionalVideos/` folder:

### Required Videos:
- `squats_professional.mp4`
- `pullups_professional.mp4`
- `shoulderpress_professional.mp4`

### Video Specifications:
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1080x1920 (portrait) or 1920x1080 (landscape)
- **Duration**: 30-60 seconds per exercise
- **Frame Rate**: 30fps minimum
- **Quality**: High definition, well-lit, clear view of form
- **Background**: Neutral/simple (easier for pose detection)

### Important:
After adding these videos, make sure to add them to the Xcode project target so they're included in the app bundle.
